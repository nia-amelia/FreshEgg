# FreshEgg

Aplikasi manajemen peternakan ayam petelur berbasis Flask dan PostgreSQL.

## Fitur

- Dashboard Statistik
- Manajemen Kandang
- Produksi Harian
- Harga Telur
- Penjualan
- Pendapatan
- Laporan Bulanan
- Export Excel
- Export PDF
- Dark Mode

## Teknologi

- Flask
- PostgreSQL (Supabase)
- Bootstrap 5
- Chart.js
- OpenPyXL
- ReportLab
